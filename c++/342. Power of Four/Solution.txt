//num & (num - 1)可以用来判断一个数是否为2的次方数，更进一步说，就是二进制表示下，只有最高位是1，那么由于是2的次方数，不一定是4的次方数，
//比如8，所以我们还要其他的限定条件，我们仔细观察可以发现，
//4的次方数的最高位的1都是计数位，那么我们只需与上一个数(0x55555555) <==> 1010101010101010101010101010101，如果得到的数还是其本身，则可以肯定其为4的次方数：
class Solution {
public:
    bool isPowerOfFour(int num) {
        return (num>0)&&(((num-1)&num)==0)&&(num&0x55555555);
    }
};
